# 📚 Taller 3 - Bootstrap 5

## 📝 Descripción

Este proyecto fue desarrollado como parte de la formación **ADSO (Análisis y Desarrollo de Software)**. Tiene como objetivo aplicar conceptos de diseño web utilizando **HTML5**, **CSS3** y **Bootstrap 5**, implementando componentes modernos, responsivos y accesibles.

## 🚀 Características

* Barra de navegación (Navbar) responsiva.
* Formularios de registro e inicio de sesión.
* Tablas con estilos de Bootstrap.
* Tarjetas informativas (Cards).
* Uso de colores y componentes modernos.
* Diseño adaptable para diferentes dispositivos.
* Buenas prácticas de estructura HTML.

## 🛠️ Tecnologías Utilizadas

* HTML5
* CSS3
* Bootstrap 5
* Git
* GitHub

## 🎯 Objetivo Académico

Fortalecer las habilidades en el desarrollo de interfaces web mediante el uso de Bootstrap 5, aplicando componentes visuales, diseño responsivo y control de versiones con Git y GitHub.
